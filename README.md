 # DealFlowPerspecties
-Página web que se usara para mobil y ordenador sobre noticias del sector privado financiero en LatAm
+
+Página web sobre noticias del sector privado financiero en LatAm.
+
+## Cómo visualizar la página
+- Archivo principal: `index.html` (en minúsculas para que GitHub Pages lo detecte).
+- Opción rápida: abre `index.html` directamente en tu navegador.
+- Opción servidor local: desde esta carpeta ejecuta `python3 -m http.server 8000` y visita `http://localhost:8000/index.html`.
+
+El sitio es responsive y usa HTML, CSS y JavaScript en un solo archivo.
